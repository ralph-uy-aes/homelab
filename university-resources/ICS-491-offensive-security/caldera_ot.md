# Caldera and Caldera for OT

#### CTI
- Cyber Threat Intelligence
    - Reporting on cyber incidents and methods real adversaries have or are using
- Extract adversary behaviors from reporting
    - Use of knowledge of adversary behaviors to inform defenders

#### Caldera
- Open Source Adversary Emulation Platform
- Automatable, repeatable emulation of realistic adversary attacks

#### Vocab
- Agent: deployed software that communicates with Caldera server
- Ability - Specific att&ck
- Adversary - Logical goruping of abilities
- Fact - Data source used
