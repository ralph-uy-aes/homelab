# Lecture 0 - Introduction

#### Course
- We will learn about roles of an OS, fundamental components, and principles of modern OS
- Assignments will be more about using the OS than implementing it
- Some lectures are inverted

#### Grades
- 70 points from quizzes (8), 600 points from midterms (2), 330 points from final (1)
- Optional homeworks 2 or 3
- 5-10 bonus points from CES

#### Exams
- Taken in class time, closed-note, pocket calculators ok
- Randomly generated, no cheating
- Midterms will have exercises that match homework and practice problems
- Final exam will have one selected exercise form each midterm

#### Quizzes
- Taken on first lecture day of the week, unless holiday
- Taken in first 10 minutes of class
- Worst grade discarded
- Go through lecture notes, think of what you might be asked during the actual quiz

#### Terminology
- This class will have a lot of terminology
- Use and recognize the right terms, part of the exams/quizzes
