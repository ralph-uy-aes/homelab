# 3.2 Firewall Types

#### The universal security control
- If you're connected to the internet, you're using a firewall
- Firewalls control the flow of network traffic
- Can be used by corporate to control outbound and inbound data
    - Prevent sensitive materials from going out
    - Or NSFW content from coming in to worksites
- Parental controls can be put on firewalls
- It is a protection against evil
    - Antivirus in itself

#### Network-based firewalls
- Filters traffic
- Traditional firewalls use OSI Layer 4 or Transport and can block by port number
- NGFW use OSI Layer 7 or application and can block by what application is connecting
- Most firewalls can also be Layer 3 devices or routers
    - Sits at the edge or ingress/egress of the network
    - Provides NAT and other dynamic routing communications

#### Unified Threat Management (UTM) or All-in-one security appliance
- Older devices that act as the primary security gateway
    - Bundles a lot of different functionality to provide security
- May have
    - URL filter
    - Content inspection
    - Malware inspection
    - Spam filter
    - CSU/DSU
    - Router/Switch capabilities
    - Firewall
    - IDS/IPS
    - Bandwidth Shaper
    - VPN endpoint
- Drawback is performance, device will slow down if you turn everything on

#### Next-Generation Firewall (NGFW)
- Uses the Application layer
- Can be called different names
    - Application layer gateway
    - Stateful multilayer inspection
    - Deep packet inspection
- Requires some advanced decodes
    - Every packet must be analyzed and categorized before a security decision is determined
- More granular
    - Instead of blocking HTTP and HTTPS entirely, you can block certain sites or services regardless of whether they share a port
    - IPS characteristics such as applying application-specific vulnerability signatures to traffic
- Content Filtering
    - URL filters
    - Control website traffic by category

#### Web Application Firewall (WAF)
- Not like a normal firewall
    - Applies rules to HTTP/HTTPS
- Allows or denies based on expected input
    - Think blocking SQL injections
- Major focus of PCI DSS
- Often used with NGFWs
