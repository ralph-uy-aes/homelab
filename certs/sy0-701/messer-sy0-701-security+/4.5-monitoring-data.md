# 4.5 Monitoring Data

#### File Integrity Monitoring (FIM)
- Some files change all the time
    - Some files should never change
- Monitor important OS and application files
    - Alert when they do
- FIM does the above
    - Can be built in
    - Windows - System File Checker (SFC)
    - Linux - Tripwire
    - Many other host-based IPS options

#### Data Loss Prevention (DLP)
- Can block sensitive data transmission before they even get sent
- Stop data leaks, stop attackers
- On your computer
    - Data in use
    - Endpoint DLP
- On your network
    - Data in motion
- On your server
    - Data at rest

#### USB Blocking
- DLP on a workstation
    - Allow or deny certain tasks
- November 2008 - US DoD
    - Worm virus "agent.btz"
        - Replicates using USB storage
        - DoD banned use of flash media and storage devices
    - All devices had to be updated
        - Local DLP agent handled USB blocking
    - Block was lifted and replaced with strict guidelines in 2010

#### Cloud-based DLP
- Located between users and internet
    - Watch every byte of network traffic
    - No hardware, no software
- Block custom defined data strings
    - Unique data for your organization
- Manage access to URLs
    - Prevent file transfers to cloud storage
- Block viruses and malware
    - Anything that traverses the network
  
#### DLP and email
- Email continues to be the most critical risk vector
    - Inbound/outbound threats
- Check every email inbound and outbound
- Inbound
    - Block keywords, identify impostors, quarantine emails
- Outbound
    - Fake wire transfers, W-2 transmissions, employee/customer information

#### Emailing a Spreadsheet Template
- November 2016
- Boeing employee sent a seemingly blank spreadsheet template, but contained 36,000 PII of Boeing employees in hidden columns
- Boeing sells its own DLP, but didn't use it here
    - They only use it in customer networks
