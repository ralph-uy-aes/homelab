# 2.4 Viruses and Worms

#### Virus
- Malware that can reproduce itself
    - Needs you to start an .exe
- Reproduces through file systems or network
- May or may not cause problems
    - Some viruses spread invisibly, some are annoying and obvious
- Anti-virus is very common
    - Thousands of new viruses every week
    - Keep your signature file updated on your antivirus, identifies malicious software

#### Virus Types
- Program Viruses
    - Part of application
- Boot Sector Viruses
    - Virus runs on boot, no need the OS
- Script Viruses
    - OS and browser based
- Macro Viruses
    - Common in Microsoft suite

#### Fileless Virus
- Stealth attack
    - Never writes software or malicious codes in to storage drives
- Avoids antivirus
- Operates in memory
    - Never installed in a file or application

#### Worms
- Malware that self-replicates
    - Doesn't need you to do anything
    - Uses the network as a transmission medium
- Can take over many systems very quickly if there's nothing to limit it
- Firewall and IDS/IPS can mitigate many worm infestations
    - Doesn't help much once the worm gets inside
- Relatively rare

#### Wannacry Worm
- Propagates automatically and installed ransomware
- Starts with infected computer
    - Searches for vulnerable systems
- Vulnerable computer is exploited with EternalBlue
    - Backdoor is installed and Wannacry is downloaded
- Repeats throughout entire network
