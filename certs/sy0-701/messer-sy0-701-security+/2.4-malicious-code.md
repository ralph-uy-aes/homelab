# 2.4 Malicious Code

#### Exploiting a vulnerability
- Many techniques
- Some don't require technical skills
- There are still other methods to get into well-secured systems
    - Exploit with Malicious code

#### Malicious Code
- Attackers use any opportunity
- Many different forms
    - Therefore protection must come from different sources

#### Malicious Code Example
- WannaCry Ransomware
    - Executable exploited vulnerability in Windows SMBv1
    - Arbitrary code execution
- British Airways XSS
    - 22 lines of malicious JS code on checkout pages
    - Information stolen from 380k victims
- Estonian Central Health Database
    - SQL injection
    - Breached all health information for Estonian citizens
