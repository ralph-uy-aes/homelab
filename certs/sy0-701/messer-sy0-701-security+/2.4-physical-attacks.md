# 2.4 Physical Attacks

#### Physical Attacks
- Old-school security
    - No keyboard, mouse, or command line
- Many different ways to circumvent digital security
    - A physical approach must be considered
- If you have physical access to a server, you have full control
    - An OS can't stop an in-person attack
- Door locks keep out the honest people
    - There's always a way in

#### Brute Force
- Physical version
    - No password required
- Push through the obstruction
    - Brawn beats brains
- Check your physical security
    - Check the windows
    - Try the doors
- Attackers will try everything
- You should be ready for anything

#### RFID Cloning
- RFID is everywhere
    - Access badges
    - Key fobs
- Duplicators on Amazon are less than $50
- Duplication process takes seconds
    - Read one card and copy to another
- This is why we have MFA
    - Use another factor with the card

#### Environmental Attacks
- Attack everything supporting the technology
- Power monitoring
    - Turn off all the power in the data center
    - Usually can be found outside of the building
- HVAC and humidity controls
    - Usually have less security
    - Large data centers must be properly cooled
    - Systems may shutdown upon overheating
- Fire suppression
    - Can be used to cause DoS
