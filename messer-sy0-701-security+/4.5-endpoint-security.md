# 4.5 Endpoint Security

#### Endpoint
- Device used by a user
    - Has their applications and data
- Stop attackers
    - Inbound and outbound
- Many different platforms
    - Mobile/desktop
- Protection is multi-faceted
    - Defense in Depth

#### Edge vs Access Control
- Control at the edge
    - Internet link
    - Managed primarily through firewall rules
    - Firewall rules rarely change
- Access Control
    - Control from wherever you are
        - In or out
    - Access can be based on many rules
        - User, group, location, time, application, etc
    - Access can be easily revoked or changed

#### Posture Assessment
- Check every system, including user's devices
- Find if anything isn't up to latest security standards or vulnerabilities
- If found, limit access
- Good time to do this is initial connection to a network
    - Is it trusted?
    - Does it have anti-virus?
    - Are there corporate applications?
    - Is it mobile?
    - Is it disk encrypted?

#### Health Checks/Posture Assessment
- Persistent agents
    - Permanently installed in a system
    - Periodic updates may be required
- Dissolvable agents
    - No formal installation required
    - Runs during posture assessment
    - Terminates itself after assessment
- Agentless NAC or Network Access Control
    - Integrated with Active Directory
    - Checks during login and logoff
    - Can't be scheduled

#### Failing your assessment
- You determine what happens if a device fails an assessment
- Do you quarantine, notify, or anything else?
- Once resolved, try again
    - May require additional fixes

#### Endpoint Detection and Response (EDR)
- Scale to meet the increasing number of threats
- Detect a threat
    - Signatures aren't the only detection tool
    - Behavioral analysis, machine learning, process monitoring
    - Lightweight agent on the endpoint
- Investigate threat
    - RCA or root cause analysis
- Respond to threat
    - Isolate system, quarantine threat, rollback
    - API driven, no user or tech required

#### Extender Detection and Response (XDR)
- Evolution of EDR
    - Able to monitor large amount of data and make connections
    - Improved on everything
    - Attacks involve more than just the endpoint
- Add network-based detection with this
    - Investigate and respond to network anomalies
- Correlate endpoint, network, and cloud data
    - Improve detection rates
    - Simplify security event investigations

#### User Behavior Analytics
- Interprets user activity
    - Their devices, network traffic, data repositories, etc
    - Builds a baseline or normal user activity
- Watches for anything unusual
    - Use a set of rules, pattern matching, statistical analysis
