# 2.3 Mobile Device Vulnerabilities

#### Mobile Device Security
- Challenging to secure
    - Often needs additional security policies and systems
- Relatively small
    - Can be almost invisible
- Always in motion
    - Never know where it might be 
- Packed with sensitive data
    - Personal and organizational
- Constantly connected to the internet

#### Jailbreaking/ Rooting
- Mobile devices are purpose-built systems
    - You don't have access to the OS
- However, OSes can be replaced 
    - Android - rooting
    - iOS - Jailbreaking
- Can be installed with custom firmware and replace the OS
- Grants uncontrolled access
    - Circumvents security features
    - MDM (Mobile Device Manager) becomes relatively useless

#### Sideloading
- Malicious apps can be a significant security concern
    - One Trojan horse can create a data breach 
- OS or device managers could manage installation sources
    - Controlling the global or local app stores
    - Restrict types of applications and installation sources
- Sideloading circumvents security
    - Apps can be installed manually without using app store
    - MDM becomes relatively useless
