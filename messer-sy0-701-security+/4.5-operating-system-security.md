# 4.5 Operating System Security

#### Active Directory
- Database containing all computers on network
    - Computers, accounts ,file shares, printers, groups, etc
    - Windows-based
- Can be used to manage authentication, perform access control, add and remove accounts, and reset passwords

#### Group Policy
- Allows us to set different configurations, settings, or policies for individual users or computers
- Can be used to manage local and domain policies
- It is a central console
    - Login Scripts
    - Network configurations (QoS)
    - Security parameters
- Comprehensive control
    - Hundreds of configuration options

#### Security-Enhanced Linux (SELinux)
- Security patches for the Linux kernel
    - Adds MAC or mandatory access control
    - Linux typically only has DAC or discretionary access control
- Limits application access
    - Concept of least privilege
    - Potential breaches have limited scope
- Open-source
    - Already included as an option in many distros
