# 3.2 Port Security

#### Port Security
- Can refer to interface port security or WAP connection security
- Commonly used on wireless networks, but also works on wired

#### Extensible Authentication Protocol (EAP)
- Authentication framework
- Manufacturers can build their own EAP methods
- EAP works with 802.1X
    - Prevents access to the network until authentication succeeds

#### IEEE 802.1X
- Port-based Network Access Control (NAC)
- You don't get access to the network until you authenticate
- Used in conjunction with other authentication database such as RADIUS, LDAP, TACACS+, Kerberos, etc

#### IEEE 802.1X and EAP
- Supplicant - the client
- Authenticator - device that provides access
- Authentication server - validates client's credentials
- Process
    - Starts with supplicant sending an initialization
    - Authenticator sends an EAP-Request
    - Supplicant responds with EAP-Response
    - Authenticator tells Auth Server about new client
    - Auth Server asks for credentials, relayed to client by Authenticator
    - Supplicant provides credentials, relayed to Auth Server by Authenticator
