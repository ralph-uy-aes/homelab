# 4.1 Securing Wireless and Mobile

#### Site Surveys
- Determine existing wireless landscape
    - Sample existing wireless spectrum
- Identify existing access points
- Work around existing frequencies
    - Layout and plan for interference
- Plan for ongoing site surveys
    - Things will certainly change
- Heat maps
    - Can be used to visualize signal strengths on network

#### Wireless Survey Tools
- Signal coverage and potential interference can be monitored with built-in tools or third-party tools
- You can also get a spectrum analyzer to see all signals on a frequency that originated from an AP or a mobile device
- See NetSpot

#### Mobile Device Management (MDM)
- Manage company-owned and/or user-owned mobile devices
    - BYOD - Bring Your Own Device
- Centralized management of mobile devices
    - Provides specialized functionality
    - Can rollout policies and required applications
- Can set policies on virtually everything on your device
    - Control the deice remotely
    - Set policies on app, data, camera, internet, etc
    - Segment user data and company data
- Manage access control
    - Force inactivity lockouts on user devices

#### BYOD
- or BYOT (technology)
- Employee owns this device
    - But they need to meet the company's requirements
- Difficult to secure
    - Both a home device and a work device
    - How is data protected?
    - What happens to the data if a user leaves or sells the device?

#### COPE
- Corporate owned, personally enabled
    - Company buys the device
    - Used as both a corporate and personal device
- Organization keeps full control of the device
    - Similar to company-owned laptops and desktops
- Information is protected using corporate policies
    - Information can be deleted at any time
- CYOD - Choose Your Own Device
    - Similar to COPE, but user picks

#### Cellular Networks
- Mobile devices are cellular phones
    - Contain a great deal of data
    - Work on 4G and 5G
- Separated in different areas of geography called "cells"
    - Antenna coverages a cell with certain frequencies
- Security Concerns
    - Traffic monitoring
    - Location tracking
    - Worldwide access to mobile devices

#### Wi-Fi
- Local network access = local security problems
- Same security concerns as other Wi-Fi devices
- Data capture
    - Encrypt your data
- On-path attack
    - Modify and/or monitor data
- Denial of Service
    - Frequency interference

#### Bluetooth
- High speed communication over short distances
    - Bluetooth is sometimes called a PAN or personal area network
- Connects mobile devices
- Do not have bluetooth on unless you're using it actively
    - Moreover, do not connect to unknown devices
- Usually has formal pairing process to prevent unauthorized connections
