# 3.2 Secure Communication

#### VPNs
- Virtual Private Networks
    - Encrypted private data traversing a public network
    - Imagine a bridge from your home network to your office network
- VPN Concentrator
    - Manages VPN connections
    - Encryption/decryption access device
    - It is often integrated with a NGFW
- Used with client software and can be built into the OS
- Many deployment options
    - Specialized cryptographic hardware
    - Software-based options available

#### Encrypted Tunnel
- How is the data encrypted and routed on the internet between different networks?
- IPsec Headers and IPsec Trailers wrap around your original packet
- Another IP Header wraps around all this entirely and is delivered to the other network where it is then decrypted

#### SSL/TLS VPN (Secure Sockets Layer/ Transport Layer Security)
- Uses common SSL/TLS protocol (tcp/443)
    - No firewall issues (almost)
- Usually remote access communication, so no big VPN clients
- Authenticates users
    - No requirement for digital certificates or shared passwords
- Can be run from a browser or from a VPN client
    - Across many OS
- On-demand access from a remote device
    - Software connects to VPN concentrator
    - Can be configured as always-on

#### Site-to-site IPsec VPN
- Always-on
- Firewalls often act as VPN concentrators
    - Probably already have firewalls in place

#### Software Defined Networking in a Wide Area Network (SD-WAN)
- Specifically designed for the cloud
- Instead of a single on-premise data center, cloud servers can be anywhere
- Cloud-based applications communicate directly to the cloud
- No need to hop through a central point

#### Secure Access Service Edge (SASE)
- Update secure access for cloud services
- A Next-generation VPN
- Security technologies are in the cloud
    - Located close to existing cloud services
- SASE clients downloaded on all devices
    - Streamlined and automatic

#### Selection of effective controls
- Many different security options
    - Selecting the right choice can be challenging
- VPN
    - SSL/TLS VPN for user access
    - IPsec tunnels for site-to-site access
- SD-WAN
    - Manage the network connectivity to the cloud
    - Does not adequately address security concerns
- SASE
    - Complete network and security solution
    - Requires planning and implementation
